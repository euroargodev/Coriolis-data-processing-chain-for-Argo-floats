Pour mettre les numéros de cycles à des fichiers bien découpés il suffit juste d'utiliser:
1- move_and_rename_argos_files
puis de contrôler le résultat avec
2- check_argos_cycle_files

Si l'on veut (doit) tout refaire les fichiers, le mieux est de reprendre la procédure complète, à savoir:
1- Mettre tous les fichiers dans le même répertoire
2- split_argos_cycles => un fichier Argos par flotteur et par passage satellite
3- delete_double_argos_split => suppression des doublons
4- create_argos_cycle_files => fichiers cycle non encore identifiés
5- co_cls_correct_argos_raw_file => correction du compteur de lignes de l'entête Argos
6- move_and_rename_argos_files => fichiers cycle identifiés (donc décodables)
7- check_argos_cycle_files.m => permet de vérifier les aspects temporels et l'attribution correcte des numéros de cycles
